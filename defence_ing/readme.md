# 对抗样本防御系统 (Adversarial Defense with Integrated Gradients)

基于 Integrated Gradients (IG) 异常检测的深度学习模型防御系统，用于检测和防御对抗样本攻击。

## 项目简介

本项目实现了一个完整的对抗样本攻击与防御评估框架，使用 PGD (Projected Gradient Descent) 攻击方法生成对抗样本，并通过 Integrated Gradients 特征归因分析来检测异常样本。

### 核心功能

- **对抗攻击**: 使用 PGD 方法生成对抗样本
- **异常检测**: 基于 Integrated Gradients 的异常检测器
- **动态阈值**: 自动校准检测阈值，平衡精度与鲁棒性
- **量化评估**: 全面的性能指标评估
- **可视化**: 生成直观的评估图表和样本对比图

## 技术架构

### 攻击方法
- **PGD (Projected Gradient Descent)**: 一种强大的白盒攻击方法
  - 扰动大小: ε = 8/255
  - 步长: α = 2/255
  - 迭代步数: 10

### 防御方法
- **Integrated Gradients (IG) 异常检测**: 
  - 计算特征归因图
  - 通过归因图方差识别异常样本
  - 动态阈值校准（默认 80% 置信区间）

### 模型与数据
- **模型**: ResNet20 (预训练于 CIFAR-10)
- **数据集**: CIFAR-10 测试集（使用 500 个样本进行评估）

## 使用方法

### 运行完整评估

```bash
python main.py
```

程序将自动执行以下步骤：

1. **加载数据与模型**: 加载 CIFAR-10 数据集和预训练 ResNet20 模型
2. **初始化攻击器与防御器**: 配置 PGD 攻击器和 IG 检测器
3. **执行评估**: 
   - 使用干净样本校准检测器阈值
   - 评估原始模型性能
   - 生成对抗样本并评估攻击效果
   - 评估防御后性能
4. **生成可视化**: 保存评估指标图表和样本对比图

### 输出结果

运行完成后将生成：

- **控制台输出**: 量化评估结果
  ```
  --- 量化评估结果 ---
  原始干净样本精度: XX.XX%
  防御后干净样本精度 (精度下降): XX.XX% (下降了 X.XX%)
  对抗攻击成功率 (原始鲁棒性): XX.XX%
  拦截后鲁棒性 (鲁棒性提升): XX.XX% (提升了 X.XX%)
  单张样本基础推理延迟: X.XX ms
  单张样本引入防御后延迟: X.XX ms (开销倍数: X.XXx)
  ```

- **evaluation_metrics.png**: 精度与鲁棒性权衡、推理延迟对比图
- **adversarial_examples.png**: 干净样本与对抗样本对比图

## 核心模块说明

### config.py
配置文件，包含：
- 设备配置（GPU/CPU）
- 批次大小
- 攻击参数（ε, α, 步数）
- 防御阈值

### attacker.py
PGD 攻击器实现，使用 `torchattacks` 库创建攻击实例。

### defender.py
IG 异常检测器实现：
- `compute_score()`: 计算特征归因和异常分数
- `calibrate()`: 基于干净样本校准动态阈值
- `detect()`: 检测异常样本

### evaluator.py
评估流程实现：
- 防御器校准
- 干净样本评估
- 对抗样本生成与评估
- 防御效果评估
- 延迟测量

### visualizer.py
可视化工具：
- `plot_metrics()`: 绘制精度、鲁棒性和延迟对比图
- `plot_images()`: 绘制干净样本与对抗样本对比图

## 性能指标

系统评估以下关键指标：

1. **干净样本精度**: 模型在干净样本上的准确率
2. **防御后干净样本精度**: 引入防御后的准确率（考虑误检）
3. **对抗攻击成功率**: 模型在对抗样本上的准确率（原始鲁棒性）
4. **拦截后鲁棒性**: 防御后模型在对抗样本上的鲁棒性
5. **推理延迟**: 基础推理和防御后推理的延迟对比

## 技术细节

### Integrated Gradients 原理

Integrated Gradients 是一种基于梯度的特征归因方法，通过计算从基线到输入的积分路径上的梯度累积来量化每个特征对模型预测的贡献。对抗样本通常会导致异常的特征归因模式，因此可以通过分析归因图的统计特性来检测异常。

### 动态阈值校准

系统使用干净样本的特征归因方差分布来计算动态阈值。默认使用 80% 分位数，意味着允许 20% 的干净样本被误判为异常，以换取更高的对抗样本检测率。
